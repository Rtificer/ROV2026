Retrieve and configure controller parameters from the ROS parameter server. 

This method is called when the controller transitions to the "{}configured"{} state. It loads all required parameters, initializes PID controllers for each degree of freedom (DOF), and sets up ROS publishers and subscribers.

The following parameters are retrieved\+: 
\begin{DoxyParams}{Parameters}
{\em dof\+\_\+names} & Vector of strings. Names of the degrees of freedom to control. \\
\hline
{\em reference\+\_\+topic} & String. Topic name for the desired velocity input. \\
\hline
{\em state\+\_\+topic} & String. Topic name for the current velocity input. \\
\hline
{\em output\+\_\+topic} & String. Topic name for the desired wrench output.\\
\hline
\end{DoxyParams}
For each DOF in {\ttfamily dof\+\_\+names}, the following PID gain parameters are also retrieved\+:
\begin{DoxyItemize}
\item gains.$<$dof$>$.p (double) Proportional gain
\item gains.$<$dof$>$.i (double) Integral gain
\item gains.$<$dof$>$.d (double) Derivative gain
\item gains.$<$dof$>$.i\+\_\+clamp\+\_\+max (double) Maximum integral clamp
\item gains.$<$dof$>$.i\+\_\+clamp\+\_\+min (double) Minimum integral clamp
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em state} & The previous lifecycle state (unused). \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Callback\+Return\+::\+SUCCESS if configuration is successful, Callback\+Return\+::\+ERROR otherwise. 
\end{DoxyReturn}

Compute and publish the desired wrench using PID controllers for each DOF. 

This method is called periodically to update the controller output. It computes the error between the reference and state for each degree of freedom (DOF), applies the corresponding PID controller, and publishes the resulting wrench.


\begin{DoxyParams}{Parameters}
{\em time} & The current time. \\
\hline
{\em period} & The time elapsed since the last update.\\
\hline
\end{DoxyParams}
Internal parameters and topics used\+:
\begin{DoxyItemize}
\item {\ttfamily ref\+\_\+} The latest reference (desired) velocity (geometry\+\_\+msgs\+::msg\+::\+Twist).
\item {\ttfamily state\+\_\+} The latest measured (current) velocity (geometry\+\_\+msgs\+::msg\+::\+Twist).
\item {\ttfamily pids\+\_\+} Vector of PID controllers, one for each DOF.
\item {\ttfamily wrench\+\_\+pub\+\_\+} Publisher for the computed wrench (geometry\+\_\+msgs\+::msg\+::\+Wrench).
\end{DoxyItemize}

\begin{DoxyReturn}{Returns}
controller\+\_\+interface\+::return\+\_\+type\+::\+OK if successful.
